# YYH 作品站图片提示词包（零素材直出版）

适用场景
- 你不给 GPT 太多资料
- 你想直接复制提示词就开始出图
- 目标是生成“和项目强相关”的作品站封面图

统一要求（建议每次都保留）
请生成一张适合作品站展示的高端产品封面图，风格统一为：premium product cover, dark minimal, clean composition, realistic software product marketing still, quiet luxury, restrained lighting, clean background, high-end website cover image。

严格要求：
- 不要像素污染
- 不要光污染
- 不要颜色污染
- 不要霓虹
- 不要赛博朋克
- 不要彩虹渐变
- 不要镜头光斑
- 不要杂乱桌面
- 不要漂浮粒子
- 不要廉价 3D 特效
- 不要假大空概念图
- 不要生成大段乱码文字
- 不要过度 UI 炫技
- 只保留真实、克制、成熟的产品感

推荐参数
- 横图
- 16:10 或 3:2
- 用于作品站封面
- 画面要有留白
- 主体必须清晰

--------------------------------------------------
直接可复制版：
生成一张高端 macOS 产品封面图。主体是一台深空灰色 MacBook Pro，屏幕顶部刘海区域被重新设计成可交互的信息中心，像“Mac 版动态岛”，自然显示音乐控制、电量状态、文件拖放提示、日历提醒等功能。画面重点是“系统级整合感”和“成熟桌面产品感”，不是概念特效。背景为纯净深黑或深石墨灰，构图克制，留白充足，设备本身高级、真实、冷静。屏幕界面要精致、可信、现代，但不能太花，不能像游戏 HUD，不能像黑客电影，不能有发光边缘和过强反射。整体像苹果级别桌面效率产品的官方网站主视觉图，适合作为高端作品站封面。
