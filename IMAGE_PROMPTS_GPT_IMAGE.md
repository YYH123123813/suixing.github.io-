# YYH 作品站图片提示词包（GPT 最新生图模型）

用途：为作品站生成统一风格的项目封面图 / 展示图
目标：高级、克制、干净、可放进当前深色极简作品站

使用方式：
1. 先上传该项目已有的真实截图 / 图标 / 线框 / 草图（如果有）
2. 再粘贴下面对应项目的 prompt
3. 如果没有真实图，也可以先纯文生图
4. 优先生成横向比例，适配作品站卡片

统一风格规则（所有项目都先加上这一段）
- 画面风格：premium product showcase, dark minimal, clean composition, realistic software product marketing still
- 背景：深石墨黑 / 深灰，不要复杂环境
- 光线：柔和、克制、单一主光源，边缘高光很轻
- 色彩：整体低饱和，只允许 1 个轻微功能性色彩点缀
- 质感：高级、冷静、可信，不要科幻感过强
- 构图：主体清晰，留白充足，适合放在作品站里
- 严格避免：像素污染、光污染、颜色污染、霓虹、赛博朋克、彩虹渐变、粒子、镜头炫光、脏纹理、过强反射、过强 bloom、杂乱桌面、廉价 3D 感、花哨 UI、假大空概念图
- 文本要求：不要生成大段假文字；如果必须出现 UI 文本，只保留极少量、模糊但整洁的界面层级

推荐比例
- 精选项目封面：16:10 或 3:2 横图
- 索引项目封面：16:10 横图
- 推荐分辨率：1792×1024 或同等横向高分辨率
